## 📌 About the Project
This is a **fully responsive single-page website** for a fictional pizza restaurant called **Gorom Pizza**.  
Built as a frontend learning project to practice modern HTML & CSS techniques (Flexbox, Grid, responsive design, hover animations, etc.).

The design focuses on:
- Eye-catching hero section
- Pizza menu showcase
- Customer testimonials
- Online order / contact form
- Clean footer

## ✨ Features
- ✅ 100% Responsive (Mobile + Tablet + Desktop)
- ✅ Modern & Clean UI with custom CSS
- ✅ Smooth hover effects and transitions
- ✅ Testimonials section
- ✅ Working-style online order form
- ✅ SEO-friendly structure
- ✅ No frameworks – pure HTML & CSS only

## 🛠️ Technologies Used
- **HTML5**
- **CSS3** (Flexbox, Grid, Custom Properties, Animations)
